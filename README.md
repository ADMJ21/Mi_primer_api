# Mi_primer_api
## Descripción
TaskFlow es una mini-plataforma construida mediante una API REST que modela operaciones típicas de un sistema real utilizando estructuras de datos lineales en memoria (Lista Enlazada, Pila y Cola).

## Stack
* **Lenguaje:** JavaScript (Node.js)
* **Framework:** Express.js
